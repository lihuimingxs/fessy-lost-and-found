# Fessy Lost & Found 失物招领系统

基于 GitHub Pages + Google Sheets 的免费失物招领平台。

## 功能
- 🔴 发布丢失信息
- 🔵 发布招领信息
- 📷 支持图片上传
- 🔍 按类型筛选
- ✅ 标记已解决

## 技术栈
- 前端：原生 HTML/CSS/JavaScript
- 后端：Google Apps Script
- 数据库：Google Sheets
- 托管：GitHub Pages

## 使用说明
1. 访问网站查看列表
2. 点击右下角 + 发布信息
3. 填写表单并提交
4. 找到物品后标记为已解决

## 维护
- 数据存储在 Google Sheets，可导出备份
- 图片建议使用 Google Drive 或 Imgur 托管
